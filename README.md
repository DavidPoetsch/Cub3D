### Run multiplayer
- install python requirements
  ```
  pip install -r test/cub_multiplayer/requirements.txt
  ```
- create `.env` file with ipaddress of your pc and other pc in `test/cub_multiplayer/.env`
  ```
  OWN_IP=x.x.x.x
  OTHER_IP=x.x.x.x
  ```
- `cd` into bonus
- run `make multi`
